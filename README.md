
# BMI Calculation in Python

Calculate your BMI using your weight and height.

## Calculate your BMI using your weight and height.

Clone the projectss

```bash
  git clone https://github.com/jskadomoto/calc_bmi_python.git
```

Navigate to the project directory

```bash
  cd calc_bmi_python/app
  python3 main.py
```
